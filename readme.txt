	Turn your website to Book Sharing Platform using this plugin in a single click. Just Click. 

== Description ==

It will make your website to a Book sharing website. It will work in a single institution 
with some features. So, Let's turn your website to a Book Sharing Website/Platform.


### Book Share Manager by Jaed Mosharraf
* [Live Demo &raquo;](http://bsm.co.nf/)
* [Get Book Share Manager &raquo;](http://bsm.co.nf/plugin)




== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin settings via WordPress Dashboard and find "<strong>Book Share Manager</strong>" activate it.<br />
